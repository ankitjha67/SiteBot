5 Messaging channels Optional

Telegram
Open Telegram, message @BotFather, send /newbot, and copy the token it gives you. The webhook is registered automatically.
Slack
At api.slack.com/apps create an app → OAuth & Permissions for the Bot Token (xoxb-…) and Basic Information for the Signing Secret. Point Event Subscriptions at the URL shown after saving.
WhatsApp (Meta)
At developers.facebook.com → your app → WhatsApp → API Setup: copy the access token and phone number ID. Choose any verify token. The app secret is under App Settings → Basic.
SMS (Twilio)
In the Twilio Console: Account SID and Auth Token on the dashboard, and a Messaging-capable phone number under Phone Numbers.
Microsoft Teams
Register an Azure Bot at portal.azure.com (Bot Framework) → copy the Microsoft App ID and a client secret (App password). Set the messaging endpoint to the URL shown after saving.
Facebook Messenger / Instagram
At developers.facebook.com → Messenger → Settings: generate a Page access token, choose a verify token, and copy the app secret from App Settings.
